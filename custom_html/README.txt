Twitch Channel Point Manager - Custom HTML Assets Folder

この「custom_html」フォルダは、ユーザー様が自由にデザインしたカスタムHTML演出ファイルを配置するためのドキュメントルートです。

【カスタム演出の作成方法】
1. 本フォルダ内に HTML、CSS、JavaScript、画像などのアセットファイルを配置します。
2. 報酬演出管理タブで「カスタムHTML演出として実行」チェックボックスをONにし、配置したHTMLファイルを選択します。
3. OBSのブラウザソースのソースURLを以下のように設定して全画面表示させます。
   例: http://localhost:28081/assets/（自動生成されたUUID）.html
   ※ 演出トリガー時に、本アプリがWebSocket経由で自動的にOBSへHTML演出の表示指令を送信します。

【JavaScriptを用いた動的演出と安全性】
セキュリティを最大限に高めるため、本アプリはローカルPC上でのPHPやPerlといった外部プログラムの実行（プロセス起動）を一切行いません。
もし演出内でTwitchの引き換えデータやランキングデータを利用したい場合は、HTML内の JavaScript（フロントエンド）から以下の安全なAPIを呼び出すことで、同等以上の高度なダイナミック演出を構築可能です。

■ ランキングデータの取得 (JSON)
URL: http://localhost:28081/api/ranking?period=0
※ JavaScript の `fetch()` を用いて非同期に最新ランキングデータを取得し、DOM操作で画面上に美麗にレンダリングすることができます。これにより、ウイルスや不正アクセスの心配を全くすることなく、100%安全かつ軽快に自由なレイアウトを構築いただけます。

【注意点】
※ セキュリティおよびパス不正防止の観点から、このフォルダの外部にあるファイルは紐づけ時にエラーとなります。アセット（画像やJS等）は必ず本フォルダ内に配置してください。
