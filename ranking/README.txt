 ranking/ フォルダ - Twitch Channel Point Manager
 統計ランキング用カスタム表示 配置フォルダ
================================================================

このフォルダは、OBS のブラウザソースに表示する
「統計ランキング」のカスタムレイアウトファイルを配置する場所です。

----------------------------------------------------------------
 ■ 対応ファイル形式
----------------------------------------------------------------

  *.html   静的HTMLファイル（CSSやJavaScriptを含めることができます）

※ セキュリティを最大限に高めるため、ローカルPC上でのPHPやPerl等のサーバーサイドスクリプト実行はサポートしておりません。
動的にデータを描画したい場合は、HTML内の JavaScript を利用してください。

----------------------------------------------------------------
 ■ OBS ブラウザソースの URL 指定方法
----------------------------------------------------------------

  http://localhost:28081/ranking/<ファイル名>

  例：
    http://localhost:28081/ranking/default.html
    http://localhost:28081/ranking/my_layout.html

  ※ ポート番号はアプリの「システム設定」→「HTTPポート」の
    設定値に合わせてください（デフォルト: 28081）

----------------------------------------------------------------
 ■ ランキングデータ取得 API
================================================================

  JavaScript から以下の API で最新のランキングデータを取得できます：

  GET http://localhost:28081/api/ranking?period=<期間>

  period の値：
    0 = 今日 (Today)
    1 = 今週 (This Week)
    2 = 今月 (This Month)
    3 = 全期間 (All Time)

  【JavaScriptでのデータ取得・描画コード例】
  -------------------------------------------------------------
  fetch('http://localhost:28081/api/ranking?period=0')
    .then(response => response.json())
    .then(data => {
        // data は [{"name": "ユーザー名", "count": 15}, ...] の配列です
        // これを使ってDOMを書き換えることで、安全かつ高速に自由なランキングを描画できます
        console.log("ランキングデータ:", data);
    });
  -------------------------------------------------------------

----------------------------------------------------------------
 ■ カスタムレイアウトのアイデア例
----------------------------------------------------------------

  ・トロフィー画像を背景に 1位・2位・3位のユーザー名を重ねて表示
  ・ユーザーのアイコンを並べた表彰台（ポディウム）デザイン
  ・スクロールするランキングテロップ
  ・週間チャンピオン専用の表彰デザイン

  CSSによるレイアウトとJavaScriptによるデータ取得だけで、
  軽量かつ100%安全に、どのようなデザインも自由に構築できます。

----------------------------------------------------------------
 ⚠ 注意事項
----------------------------------------------------------------

  ・このフォルダ「外」のファイルにはアクセスできません（パス・トラバーサル保護機能搭載）
  ・HTML内のアセット（CSS、JS、画像等）もこのフォルダ内に配置してください。
