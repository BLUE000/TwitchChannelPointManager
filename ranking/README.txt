 ranking/ フォルダ - Twitch Channel Point Manager
 統計ランキング用カスタム表示スクリプト 配置フォルダ
================================================================

このフォルダは、OBS のブラウザソースに表示する
「統計ランキング」のカスタムレイアウトファイルを配置する場所です。

----------------------------------------------------------------
 ■ 対応ファイル形式
----------------------------------------------------------------

  *.html   静的HTMLファイル（そのままブラウザに配信されます）
  *.php    PHPスクリプト（実行結果がHTTPレスポンスになります）
  *.pl     Perlスクリプト（実行結果がHTTPレスポンスになります）
  *.cgi    Perlスクリプト（同上）

PHP・Perl を使用するには、アプリの「システム設定」タブで
外部スクリプト連携機能を有効にし、インタープリターのパスを
設定してください。

----------------------------------------------------------------
 ■ OBS ブラウザソースの URL 指定方法
----------------------------------------------------------------

  http://localhost:28081/ranking/<ファイル名>

  例：
    http://localhost:28081/ranking/default.html
    http://localhost:28081/ranking/my_layout.html
    http://localhost:28081/ranking/trophy_board.php
    http://localhost:28081/ranking/summary.cgi

  ※ ポート番号はアプリの「システム設定」→「HTTPポート」の
    設定値に合わせてください（デフォルト: 28081）

----------------------------------------------------------------
 ■ ランキングデータ取得 API
----------------------------------------------------------------

  PHP・HTML 内から以下の API でランキングデータを取得できます：

  GET http://localhost:28081/api/ranking?period=<期間>

  period の値：
    0 = 今日 (Today)
    1 = 今週 (This Week)
    2 = 今月 (This Month)
    3 = 全期間 (All Time)

  レスポンス例（JSON）：
  [
    {"name": "viewer_A", "count": 15},
    {"name": "viewer_B", "count": 9},
    ...
  ]

----------------------------------------------------------------
 ■ カスタムレイアウトのアイデア例
----------------------------------------------------------------

  ・トロフィー画像を背景に 1位・2位・3位のユーザー名を表示
  ・ユーザーのアイコンを並べたポディウム演出
  ・スクロールするランキングテロップ
  ・週間チャンピオン専用の表彰デザイン
  ・PHPでデータを加工して、OBSのGDI+テキストソース用ファイルへ出力

  ランキングデータの取得・表示はすべて自由にカスタマイズできます。

----------------------------------------------------------------
 ⚠ 注意事項
----------------------------------------------------------------

  ・このフォルダ「外」のファイルは配信されません（セキュリティ保護）
  ・PHP・Perl スクリプトの実行タイムアウトは 30 秒です
  ・第三者が作成したスクリプトを実行する場合は十分ご注意ください
    スクリプトによるいかなる損害も、ツール作者は保証できません
    （すべて自己責任でご利用ください）
